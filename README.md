# Space Expedition

## Overview

Space Expedition is a C# console application for managing a space expedition artifact inventory.

This project allows the user to load artifact data, view the inventory, search for artifacts, sort items, add new artifacts, and save updated information. I chose this project because it shows my practice with C# classes, file input/output, arrays/lists, menu systems, sorting, searching, and basic object-oriented programming.

## Features

- Load artifact inventory data from a file
- Display artifact information in the console
- Search for artifacts
- Sort artifact inventory
- Add new artifact records
- Save updated inventory data
- Decode artifact names using custom logic
- Simple console menu for user interaction
